== Description ==
Via deze plugin toon je de laatste sample sales op je site. De data wordt automatisch geupdate vanop [Stockverkopen.nl](http://www.stockverkopen.nl "Stockverkopen.nl"). Je kan via een tag die je op een post of page plaatst, kiezen waar je de gegevens laat zien.  Je kan de gegevens ook tonen via een sidebar widget.

Je kan tussen de gegevens ads plaatsen indien u dit wenst. Zo kan u de inhoud monetiseren. De look and feel kan je aanpassen via stylesheets.


== Installation ==
Installeer de plugin en activeer hem
Om nu sample sales op een pagina of blogpost te tonen zet je de speciale code `__SAMPLESALES_NL__` (zonder quotes) op de plaats waar je op de pagina/blogpost de sample sales wil tonen. Meer info vind je bij Settings->Sample sales op je site (NL)
